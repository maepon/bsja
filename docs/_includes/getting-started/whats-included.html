Precompiled Bootstrap

[Once downloaded, unzip the compressed folder to see the structure of (the compiled) Bootstrap.]ダウンロード後、解答すると（コンパイル済みの）Bootstrapのフォルダーが確認できます。 [You'll see something like this]このような感じで見られます。:

{% highlight bash %} bootstrap/ ├── css/ │ ├── bootstrap.css │ ├── bootstrap.css.map │ ├── bootstrap.min.css │ ├── bootstrap-theme.css │ ├── bootstrap-theme.css.map │ └── bootstrap-theme.min.css ├── js/ │ ├── bootstrap.js │ └── bootstrap.min.js └── fonts/ ├── glyphicons-halflings-regular.eot ├── glyphicons-halflings-regular.svg ├── glyphicons-halflings-regular.ttf ├── glyphicons-halflings-regular.woff └── glyphicons-halflings-regular.woff2 {% endhighlight %}

コンパイルされたCSSとJS(bootstrap.*)と同様にコンパイル後にミニファイされたCSSとJS(bootstrap.min.*)を提供しています。CSSのsource maps (bootstrap.*.map)は対応しているブラウザのデベロッパーツールに向けて提供されています。Bootstrapテーマの付属品としてGlyphiconsのフォントが含まれています。

Bootstrapのソースコード

[The Bootstrap source code download includes the precompiled CSS, JavaScript, and font assets, along with source Less, JavaScript, and documentation.]ダウンロードしたBootstrapのソースコードはプリコンパイルされたCSSろJavaScriptとフォント素材に加えて、Lessのソース、JavaScriptそしてドキュメントが含まれています。 [More specifically, it includes the following and more]具体的には下記のものなどを含んでいます:

{% highlight bash %} bootstrap/ ├── less/ ├── js/ ├── fonts/ ├── dist/ │ ├── css/ │ ├── js/ │ └── fonts/ └── docs/ └── examples/ {% endhighlight %}

[The less/, js/, and fonts/ are the source code for our CSS, JS, and icon fonts (respectively).]less/、 js/、そして fonts/は、CSS、JS、そしてアイコンフォントのソースコードです。 [The dist/ folder includes everything listed in the precompiled download section above.]dist/フォルダは、上記のダウンロードの項に記載されているもののプリコンパイルされたものが含まれています。 [The docs/ folder includes the source code for our documentation, and examples/ of Bootstrap usage.]docs/フォルダはドキュメントの、examples/フォルダには利用法のソースコードが含まれています。 [Beyond that, any other included file provides support for packages, license information, and development.]その他、パッケージへの対応、ライセンスの情報、開発上のファイルなどが含まれています。
