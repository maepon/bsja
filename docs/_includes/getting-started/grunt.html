[Compiling CSS and JavaScript]CSSとJavaScriptのコンパイル

[Bootstrap uses Grunt for its build system, with convenient methods for working with the framework.]Bootstrapはフレームワークを有効に活用できるように便利な機能と一緒にビルドシステムとしてGruntを利用しています。 [It's how we compile our code, run tests, and more.]コードのコンパイル、テストの走らせ方、その他についての記載しています。

[Installing Grunt]Gruntのインストール

[To install Grunt, you must first download and install node.js (which includes npm).]Gruntをインストールするにはまずnode.jsをダウンロードしてインストールします（npmが含まれています）。 [npm stands for node packaged modules and is a way to manage development dependencies through node.js.]npmはnode packaged modulesの略語で、node.jsの開発上の依存性を管理します。

[Then, from the command line]その後、コマンドラインから:
  1. [Install grunt-cli globally with npm install -g grunt-cli.]npm install -g grunt-cliコマンドでgrunt-cliをグローバルでインストールします。
  2. [Navigate to the root /bootstrap/ directory, then run npm install.]/bootstrap/ディレクトリに移動し、npm installコマンドを走らせます。 [npm will look at the package.json file and automatically install the necessary local dependencies listed there.]npmがpackage.jsonファイルを読み込み、記載されている必要なパッケージをインストールします。

[When completed, you'll be able to run the various Grunt commands provided from the command line.]インストールが完了すると、コマンドラインから様々なGruntのコマンドが実行できます。

[Available Grunt commands]利用できるGruntコマンド

grunt dist [(Just compile CSS and JavaScript)]（CSSとJSのコンパイルのみを行います。）

[Regenerates the /dist/ directory with compiled and minified CSS and JavaScript files.]/dist/ディレクトリにコンパイルとミニファイを行ったCSSとJavaScriptのファイルを再生成します。 [As a Bootstrap user, this is normally the command you want.]Bootstrapユーザーの一般的な用途です。

grunt watch [(Watch)]（監視）

[Watches the Less source files and automatically recompiles them to CSS whenever you save a change.]Lessのソースファイルを監視し、変更が保存されると自動的にオースファイルに基づいてCSSを再コンパイルします。

grunt test [(Run tests)]（テスト実行）

[Runs JSHint and runs the QUnit tests headlessly in PhantomJS.]JSHintを実行し、PhantomJS上のQUnitでヘッドレステストを行います。

grunt docs [(Build & test the docs assets)]（ドキュメントの構成要素のビルドとテストを行う）

[Builds and tests CSS, JavaScript, and other assets which are used when running the documentation locally via jekyll serve.]jekyll serveからドキュメントを実行する際に必要なCSS、JavaScript、その他必要なファイルのビルドとテストを行います。

grunt [(Build absolutely everything and run tests)]（無条件に全てのビルドし、テストを行います）

[Compiles and minifies CSS and JavaScript, builds the documentation website, runs the HTML5 validator against the docs, regenerates the Customizer assets, and more.]CSSとJavaScriptのコンパイルとミニファイを行い、ドキュメントのWebサイトを生成し、HTML5の文法チェックをドキュメントに対して行い、関連ファイルを再生成、などを行います。 [Requires Jekyll.]Jekyllが必要です。 [Usually only necessary if you're hacking on Bootstrap itself.]Bootstrap自体のハックを行う場合は、通常このコマンドのみが必要となります。

[Troubleshooting]トラブルシューティング

[Should you encounter problems with installing dependencies or running Grunt commands, first delete the /node_modules/ directory generated by npm.]依存パッケージのインストール時やGruntコマンドの実行時にしばしば問題に遭遇する場合、まずnpmに生成された/node_modules/ディレクトリを削除してください。 [Then, rerun npm install.]その後、npm installを再度実行してください。
